# 成果管理系统功能测试用例

## 测试范围

本用例表覆盖角色 B 当前阶段需要准备的核心功能、边界值和越权操作测试。当前阶段已通过 Playwright mock E2E 先行验证可自动化的用例；真实环境吞吐量、真实账号权限矩阵、数据库一致性和最终截图留档仍需待 A 完成部署后复验。

## 测试前置条件

- 已部署前端和后端，并确认前端接口前缀，例如 `/dev-api`。
- 已准备管理员、学生、教师、学院审核员、学校审核员等测试账号。
- 已导入基础字典、部门、用户、赛事、届次、成果和权限菜单数据。
- 浏览器控制台无阻断性 JavaScript 报错。
